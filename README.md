Project Name: Fable Client
Project Purpose
Fable is a digital ecosystem designed to bridge the gap between creative writers and enthusiastic readers. The primary objective is to democratize literature by providing a secure, user-friendly platform where writers can self-publish their work and readers can explore, purchase, and read original ebooks. By integrating role-based access, secure payment gateways, and analytical dashboards, Fable transforms the traditional bookstore experience into a seamless, modern, and interactive online community.
Key Features (client)
Role-Based Dynamic Dashboard: A centralized control hub where users, writers, and admins have customized views based on their roles.
Advanced Discovery Engine: A robust "Browse Ebooks" page featuring real-time search, multi-layered filtering (by genre, price, availability), and sorting (newest, price range).
Secure Payment Integration: Seamless integration with Stripe Checkout for purchasing ebooks, including post-payment status updates and transaction tracking.
