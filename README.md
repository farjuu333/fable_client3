Project Purpose
Fable is a digital ecosystem designed to bridge the gap between creative writers and enthusiastic readers. The primary objective is to democratize literature by providing a secure, user-friendly platform where writers can self-publish their work and readers can explore, purchase, and read original ebooks. By integrating role-based access, secure payment gateways, and analytical dashboards, Fable transforms the traditional bookstore experience into a seamless, modern, and interactive online community.
Key Features (client)
Role-Based Dynamic Dashboard: A centralized control hub where users, writers, and admins have customized views based on their roles.
Advanced Discovery Engine: A robust "Browse Ebooks" page featuring real-time search, multi-layered filtering (by genre, price, availability), and sorting (newest, price range).
Secure Payment Integration: Seamless integration with Stripe Checkout for purchasing ebooks, including post-payment status updates and transaction tracking.
Interactive Ebook Details: A dedicated page for each ebook with a clean UI, displaying comprehensive information, and enabling actions like purchasing or bookmarking.
Writer's Creative Suite: An intuitive form-based interface for writers to manage their portfolio, upload covers (via ImgBB), and toggle between published/unpublished status.
Admin Analytics & Control: Visual representation of data using charts (sales, revenue) and administrative tools to manage users, roles, and content.
Technologies Used (client)
Framework: Next.js (App Router for optimized performance and routing)
Styling: Tailwind CSS (for a modern, responsive, and customizable design)
Icons & Components: Lucide React / React Icons, Shadcn UI / DaisyUI (for high-quality UI components)
Animation: Framer Motion (for smooth transitions and hover effects)
State Management: TanStack Query (React Query) for efficient data fetching and caching
Forms: React Hook Form with Zod (for validation)
Authentication: BetterAuth / NextAuth (for secure session handling)
Charts: Recharts / Chart.js (for dashboard analytics)
Image Handling: Axios (for API communication) & ImgBB API
Payment Frontend: Stripe.js (for secure payment processing)
